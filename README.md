# 🎬 YouTube字幕翻译 v3.0

**暂停视频，自动翻译字幕**

## 特点

- ⏸️ **暂停翻译** - 暂停视频自动显示翻译
- 🔄 **实时检测** - 自动检测YouTube页面
- 🌐 **多语言** - 支持中文/日文/韩文/英文
- 🎨 **简洁UI** - 暗色主题，清晰显示

## 安装

1. 打开 `chrome://extensions`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `v3` 文件夹

## 使用

1. 打开YouTube视频
2. 点击插件图标，点击「开启翻译」
3. 暂停视频，自动显示中文字幕
